# 🕵️‍♂️ 新币发行时间查询渠道大全 (Alpha Sources)

想要第一时间埋伏新币 (打新/撸毛)，你需要关注以下几个维度的信息源：

## 1. 综合聚合平台 (最全、最直观)
适合查看已经确定了发行时间的项目。

*   **CryptoRank (强烈推荐)**: [https://cryptorank.io/upcoming-ico](https://cryptorank.io/upcoming-ico)
    *   **优点**: 数据最全，包含 IDO/IEO/ICO 所有类型，还能看到融资背景和投资机构。
    *   **用法**: 关注 "Upcoming" (即将到来) 和 "Active" (进行中) 的项目。
*   **CoinMarketCap (CMC)**: [https://coinmarketcap.com/ico-calendar/](https://coinmarketcap.com/ico-calendar/)
    *   **优点**: 流量大，收录的项目通常比较正规。
    *   **缺点**: 更新速度不如 CryptoRank 快。
*   **CoinGecko**: [https://www.coingecko.com/en/ico](https://www.coingecko.com/en/ico)
    *   **特点**: 类似 CMC，可以对比使用。

## 2. 交易所官方发射台 (IEO - 质量最高)
能上大所 Launchpad 的项目，通常经过了严格筛选，暴富效应最强 (比如 Binance 的 Pixel, Portal 等)。

*   **Binance Launchpool/Launchpad**: [https://launchpad.binance.com/](https://launchpad.binance.com/)
    *   **必看**: 只要币安发公告，必是精品。
*   **OKX Jumpstart**: [https://www.okx.com/jumpstart](https://www.okx.com/jumpstart)
    *   **特点**: 质押 OKB 或 BTC 挖新币，收益也很可观。
*   **Bybit Launchpad**: [https://www.bybit.com/trade/spot/launchpad](https://www.bybit.com/trade/spot/launchpad)
    *   **特点**: 上新频率高，门槛相对较低。

## 3. 链上发射平台 (IDO - 早期机会)
这里是“百倍币”和“归零币”的诞生地，风险高，收益大。

*   **DAO Maker**: [https://daomaker.com/](https://daomaker.com/) (老牌，质量较高)
*   **Polkastarter**: [https://polkastarter.com/](https://polkastarter.com/)
*   **Seedify**: [https://seedify.fund/](https://seedify.fund/) (专注于链游/GameFi)
*   **Fjord Foundry**: [https://fjordfoundry.com/](https://fjordfoundry.com/) (LBP 荷兰拍，公平发射)

## 4. 链上实时监控 (土狗/Meme 专用)
如果你想冲刚开盘 5 分钟的“土狗币” (Meme)，看这里：

*   **DEX Screener**: [https://dexscreener.com/](https://dexscreener.com/) -> 点击 "New Pairs" (新交易对)
*   **DEXTools**: [https://www.dextools.io/](https://www.dextools.io/) -> 点击 "Live New Pairs"
*   **Ave.ai**: 手机端最好用的链上看线工具。

## 5. 社交媒体 (Alpha 猎手)
很多消息是推特先发，然后才上日历。

*   **Twitter (X)**: 关注项目官推，打开小铃铛。
*   **关键 KOL**: 关注像 `Lookonchain` (链上侦探), `Dovey Wan` (一级市场大佬) 等。
*   **Telegram**: 很多项目的官方公告群 (Announcement Channel) 是最快的。

---

### 💡 建议的操作流
1.  **每天早上**: 刷一眼 **CryptoRank** 的 "Upcoming"，看看近期有什么新项目。
2.  **重点关注**: 如果看到有 **Binance Labs**, **a16z**, **Paradigm** 投资的项目 (如 AZTEC, ESP)，加入他们的 Discord/Telegram 蹲守。
3.  **IEO 必打**: 只要币安/OKX 有活动，无脑参加。
